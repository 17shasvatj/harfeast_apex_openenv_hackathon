# W&B export and plots

Generated by `scripts/export_wandb_plots.py`.

## Files

- **wandb_reward_curve.png** — Training reward curve (mean rubric score vs step, 14-step MA).
- **wandb_loss_curve.png** — GRPO loss over gradient steps.
- **wandb_epoch_mean_reward.png** — Mean reward per epoch.
- **wandb_epoch_signal.png** — Fraction of tasks with nonzero reward and with variance (learning signal) per epoch.
- **wandb_reward_variance.png** — Reward variance over training.
- **wandb_per_task_rewards.png** — Per-task reward over steps (excludes task_08, task_13, task_14).
- **run_history.csv** — Raw step/epoch data used for the plots.

## project.csv (W&B only)

To get `project.csv` (run summaries and configs from Weights & Biases), run **without** `--local` after logging in:

```bash
wandb login   # or set env WANDB_API_KEY=your_key
python scripts/export_wandb_plots.py
```

This will overwrite the plots with data from the latest W&B run and create `project.csv`.

## Local-only (no W&B)

Plots in this folder were generated from local `ckpt_v4_q25/train_log.jsonl`:

```bash
python scripts/export_wandb_plots.py --local
```

Optionally specify a different log: `--log path/to/train_log.jsonl`.
